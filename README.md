# Проект: Автоматизация загрузки видеоконтента в TikTok

**Январь 2025 | Фриланс**

- Реализовал модуль автоматической загрузки видео через SeleniumBase с использованием режима Undetected Chromedriver для снижения вероятности детектирования антибот-защитой платформы.
- Настроил механизм авторизации через предварительно сохранённые куки, что позволило исключить повторную ручную авторизацию и ускорить процесс массовой загрузки контента.
- Реализовал поддержку прокси-серверов на уровне драйвера для работы с разными аккаунтами и минимизации риска блокировок.
- Обеспечил стабильную работу с интерфейсом TikTok, включая загрузку файлов, работу с редактируемыми элементами и управление DOM через ожидания и инъекции JavaScript.
- Реализовал обработку ошибок и корректное завершение сессии браузера.

### Стек
`Python`, `SeleniumBase`, `Undetected Chromedriver`, `Selenium`, `WebDriver`, `proxy`, `cookies`
